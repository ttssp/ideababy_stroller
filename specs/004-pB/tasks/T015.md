### R2 H1 新增: MonthlyScheduler pause hook
```python
class MonthlyScheduler:
    _paused: bool = False  # R2 H1

    def pause(self) -> None: self._paused = True
    def resume(self) -> None: self._paused = False
    def is_paused(self) -> bool: return self._paused

    async def run_monthly_job(self):
        if self._paused:
            logger.info("MonthlyScheduler is paused, skipping this run")
            return
        await monthly_review_service.generate(month_id=...)
```

T020 / T022 在 B-lite engage 时直接调 `monthly_scheduler.pause()` (R2 H1).
