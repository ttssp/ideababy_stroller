## Goal
季度学习检查 (PRD §S4 / O4 / R7 / spec.md C19): 防"学习假装". Q7 默认假设 = LLM
从笔记 wiki 抽取概念列表 + 自动出题, human 写答案 + 自评.
- `LearningCheckService.start_quarter(quarter_id)`: 调 LLMClient 从过去 90 天笔记
  + decisions 中 reason 抽 ≥ 7 个金融概念 (Q7), 给每个概念出题 (e.g. "用 ≤ 30 字
  解释 P/E ratio") + 标准答案
- Web UI `/learning`: 列出当前季度题目, human 提交答案
- 提交后 `scripts/grade_learning.py` 用 LLMClient 比对 human 答案 vs 标准答案,
  打 0-10 分, 整体均分 ≥ 7/10 视为通过 (O4 verification)
- CLI: `python -m scripts.learning_check --quarter Q1` 触发本季出题
- UI 提醒: 即使 human 想跳过, 季度到期前 nav 显示红色提醒 (non-goals D9)

## Known gotchas
- LLM 抽取数量不稳, 用 tool use 强制 exactly 7-10 个 (schema 数组长度约束)
- 评分主观性强, 用 LLM 给出"主旨匹配度", 不强求精确字面
- non-goals D9: 不能因"用了 3 个月就假设懂", UI 不能让 human 一键跳过 (要走完答题)
