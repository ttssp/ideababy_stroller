## Goal (R3 简化版, 5h)
季度学习检查 (PRD §S4 / O4 / R7 / spec.md C19): 防"学习假装". **Q7 R3 修订: human 手编 master checklist**.
- human 维护 `docs/learning_checklist.yaml` (10 概念 + 标准答案), ship 时手编一次
- `LearningCheckService.start_quarter(quarter_id)`: 从 yaml 随机抽 7 个概念, 入 quarter quiz
- Web UI `/learning`: 列出当前季度题目, human 提交答案
- 提交后 `scripts/grade_learning.py` 用 LLMClient 比对 human 答案 vs yaml 标准答案,
  打 0-10 分, 整体均分 ≥ 7/10 视为通过 (O4 verification)
- CLI: `python -m scripts.learning_check --quarter Q1` 触发本季抽题
- ~~UI 提醒~~ (R3 砍, 改为 docs 提醒)
