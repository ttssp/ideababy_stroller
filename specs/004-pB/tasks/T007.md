## Goal
本 task 是架构枢纽 (architecture.md §3.1). 实现:
1. `StrategyModule` Protocol (R2 修订: **唯一方法 `analyze()` + `source_id` 字段**, 不再
   有 conflict_resolve / devil_advocate; 架构锁死, **不可增减**, TECH-4 mitigation)
2. 三个占位实现 (R2 修订: 仅 analyze, 三 lane 平权, 无总结者):
   - `AdvisorStrategy` (source_id="advisor"): 取本周咨询师对该 ticker 的观点, LLM
     prompt 转 StrategySignal; 无观点 confidence=0.0, direction="no_view"
   - `PlaceholderModelStrategy` (source_id="placeholder_model"): Q6 默认假设 = 永远
     no_view (confidence=0.0), 不假装"私有模型有意见"
   - `AgentSynthesisStrategy` (source_id="agent_synthesis"): 用 advisor_report +
     portfolio + env_snapshot 综合 LLM 出 signal; **不读其他 lane 输出** (R9 解耦);
     **R2: 与其他两个 lane 平权**, 不再承担 conflict_resolve 主体职责
3. `StrategyRegistry`: 用 FastAPI Depends 风格的依赖注入, 列出所有已注册 module,
   每个独立可替换 (R9 / 不变量 §9.6: 不直接读他模块状态); 预留
   `register_custom(CustomStrategyModule)` 扩展位 (D7)
4. **不变量约束**: `StrategySignal.rationale_plain` 永远非空 (R3 / OP-7)
5. **R6 反诱导**: agent_synthesis prompt 中明确"≥ 50% analyze 输出建议不动",
   prompt 模板单测覆盖 (sanity check 移到 lane 内, 不再依赖 conflict_resolve)
6. **R2 新增**: lane 隔离单测 — 每个 lane 单元测试不能 mock/import 其他 lane (R9 工程化)

  @runtime_checkable
  class StrategyModule(Protocol):
      source_id: str

      def __init__(
          self,
          llm_client: LLMClient,
          source_data_provider: SourceDataProvider,
      ) -> None: ...

          portfolio: Portfolio,
