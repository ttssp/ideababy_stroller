## Known gotchas

- GitHub Actions 的 CI 运行时限:应保持 < 15min(`pytest -m "not slow and not gpu"` 不含真 LoRA,预期 < 5min)
- `uv sync --frozen` 在 CI 可能因缺 bitsandbytes CUDA wheel 失败 → CI 上 `uv sync --no-optional` 或 extra = lite
- pip-audit 若有已知 High 漏洞阻塞 CI:处理方式文档化在 CONTRIBUTING.md(评估 + whitelist with justification)
- 不用 `actions/checkout@v3`,用 `@v4`(2026 当前)
