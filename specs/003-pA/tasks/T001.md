   - `[project]`:name=recallkit, version=0.1.0.dev0, python=">=3.12,<3.13"
   - `[project.scripts]`:`pars = "pars.cli.main:app"`
   - `[project.optional-dependencies]` 分 `runtime` / `dev` 两组(先占位,T002 填实际依赖)
2. `.gitignore` 含 Python 常规 + 本项目特有(`runs/`, `checkpoints/`, `.worktrees/`, `.env*`, `worker_claude_dir/.cache/`)
3. `pars/__init__.py` 仅 `__version__` + `__all__ = []`
4. `README.md` 占位含标题、1 段介绍、"本项目处于 alpha 开发中" 声明、最小硬件要求 placeholder
5. 跑 `uv sync` 验证基础骨架可用
