季度学习检查

每季抽 7 题, 防"学习假装" (PRD §S4 / O4 / R7)。

使用流程 (CLI 驱动)

  1. 抽题: python -m scripts.learning_check --quarter 2026-Q2 — 从 docs/learning_checklist.yaml 随机抽 7 题入库。
  2. 答题: 浏览器打开 /learning/<quarter_id> — e.g. /learning/2026-Q2。
  3. 评分: python -m scripts.grade_learning --quarter 2026-Q2 — LLM 比对答案 vs 标准答案, 0-10 分。
  4. 结果: 均分 ≥ 7/10 视为通过 (O4 verification)。 刷新季度页查看 scores。

R3 简化说明

  • master checklist 由 human 手编 docs/learning_checklist.yaml (10 概念 + 标准答案)。
  • service 路径零 LLM 调用 — 仅评分脚本调 LLM 一次/季度。
  • 不做 nav 红色提醒 — 改为 docs 自查 / runbook 引用。
  • 下一次提醒由 T020 alert banner 在 90 天后自动出现。
