## 1. The idea, unpacked

### 谁是用户

不是一个"零售投资者"。是一个非常具体的人: **35-50 岁、受过硬核理工训练(ML 方向 CS PhD)、过去十几年靠上班挣工资、现在突然发现必须对家庭财务总负责的中年人**。他不是华尔街人, 但他也不是小白 —— 他理解模型、理解噪声、理解"p < 0.05 骗不了多头"这类话。他有两个家庭约束压在肩上(经济下行 + 对未来秩序的恐惧), 一个付费订阅的人类投顾(哥大背景, 每周给观点)作为 **anchor**, 一个明确的覆盖面(美股/港股/A股 × 30-50 只股票), 一个硬预算(每年 $500 数据费), 一个诚实的自我定位("不做发大财梦, 能吃波动就知足")。

这个画像独特: 绝大多数零售投资产品面向的是信息更稀缺、判断力更弱、时间更多的用户。这个人**信息不缺、判断力强、时间极稀缺**, 而且他**已经付钱买了一个人类专家**。他要的不是"AI 比人类顾问强", 他要的是"AI 能把我+人类顾问+市场信号这三方的产出, 统一到我能用的决策流里"。

### 没有这个 agent 的日常

早上起床, 他打开投顾微信群, 刷周一早报。中午开会间隙看一眼美股盘前, 记一下 NVDA 又 +3%。晚上回家陪娃, 娃睡后打开雪球 / 长桥, 看个股技术面 + 新闻, 想做决定又拿不准, 翻翻那位顾问的历史推送找依据, 找不到就刷 Twitter / Substack 碎片信息, **决定拖到明天**。他花了 1 小时, 消化了信息, 但**没留下任何决策档案, 下周的自己没办法回看今天想了什么**。每一次决策是新的, 经验不累加。

### 有了这个 agent 的日常

同样晚上 30 分钟窗口。他对 agent 说"我在想加仓 TSM"。agent 查他上月的持仓快照(单持仓集中度 / 地区暴露 / 相关性), 查他订阅的哥大顾问这周对半导体板块的立场, 查他 30 只关注股里和 TSM 相关性 > 0.6 的那几只的最近走势, **然后让他先回答三个问题**: (a) 你这次加仓占组合多少比例, 触发你自定的哪些规则? (b) 你上个月 9 号写下的 TSM 多空推理里, 哪一条仍成立、哪一条证伪了? (c) 那位顾问这周的观点和你的想法对齐吗, 如果相反你会怎么做? 他回答完, agent 产出一张单子: 本次操作的 pre-mortem 档案 + 一条给未来自己的 note ("如果一周内 TSM 跌 5%, 回来看这个档案"). **决策留下痕迹, 经验可积累**。

### 第一次打开的 30 秒 aha

第一次打开, agent 不问他要 API key, 不让他连券商。先问他两件事: "你现在 30-50 只关注股的清单发我一下; 你订阅的那位顾问的最近 4 期周报截图给我看。" 5 分钟后, agent 给他看一张**他自己也没真正看过的矩阵**: 30 只股里, 按顾问推荐等级 × 你当前持仓 × 本周技术信号分层。他会"哦", 因为他第一次看见**自己已经在追踪的东西, 被整理成了一个可决策的视图**。这个 aha 不是"AI 好强", 是"**原来我自己已经走了一半了, 只是从没把它拼起来**"。

### 6 个月后熟练期的样子

半年后, 他已经养成了一个晚间仪式: 20 分钟。前 10 分钟看"今日家庭财务 snapshot"(整个家庭跨市场净值 + 现金流状态 + 本月信号), 后 10 分钟做"决策日志维护"(本周若干个潜在 trade 的 pre-mortem 更新 + 上月 trade 的 post-mortem)。他开始**看见自己的模式**: "我过去六个月的 15 次 trade 里, 9 次在下单前没写 pre-mortem 的那些, 平均 return 比写了的低 4%。" 这个数据不是理论, 是他自己的历史。agent 变成了他的**第二记忆**。

### 100 万人用它的世界

100 万中年技术人用类似的东西, 它会不会变成一个独立品类? 大概率不会以"AI 投顾"的形态 —— 那品类已被 Robinhood / Magnifi 占据。它会以**"投资决策档案+个人化顾问连接器"**的形态出现: 每个人的 agent 是一个私人助理, 但它同时是**那位人类顾问的"观点分发引擎"**——顾问写一次推送, 触达 N 个订阅者的 agent, 每个 agent 按订阅者的组合+风险偏好把推送翻译成"这对你具体意味着什么"。人类顾问的单点价值被放大 10 倍, 订阅者得到定制化建议, 而 agent 厂商卖的不是 AI, 是"把专家变成每个粉丝的私人顾问"的基础设施。

## 2. What's genuinely novel

这个 idea 的核心 novelty 不在"AI 投资", 那已经被做烂了。真正新的是**框定方式**: 它是一个**把"人类付费顾问 + 自己的判断 + 多源数据"三方拼合的个人化决策工作台**, 不是一个"取代人类"的 robo-advisor。

市面上的 robo-advisor (Wealthfront, Betterment) 思路是"AI 替代专家"; AI 投资助手 (Magnifi, Kavout) 思路是"AI 超越散户"; 交易日志 (TradersSync, Edgewonk) 思路是"事后复盘"。**这三类都忽略了一个现实: 很多中产技术人已经在付费订阅某一个人类专家, 而且他们不想离开这个专家 —— 他们想要的是一个能把这个专家的价值放大的工具**。

如果说有 novelty, 那就是**"以订阅的人类顾问为 anchor, 以 AI 为放大器, 以个人决策档案为资产"的三位一体结构**。这在市面上没有现成对应物。

不过要诚实: 这个 novelty 更多在"组合 + 受众切片"的精确度, 而不在"算法/AI 技术的突破"。它是**执行驱动的 novelty**, 不是概念驱动的。如果只看"AI 帮你投资"这一层, 它不新。如果看"为付费订阅人类投顾的中年技术人做一个决策放大器", 那空间几乎是空的。

## 3. Utility — 三个具体使用场景

### 场景 A: 周一早晨, NVDA 财报前夜
早上 6:50, agent 主动推送: "本周 NVDA 财报周三盘后。你持仓 NVDA 占组合 7.3%, 上次你写的多头理由: '数据中心 capex 加速'; 上周你订阅顾问的观点: '短期已 price-in 部分乐观'。财报前你的三个备案: (1) 不动 —— 长逻辑未变, (2) 减半 —— 兑现部分利润, (3) 加仓 —— 持续看多。选择一个, 写下 if-then 触发条件, 我帮你记下来。" 他 30 秒内选了 (2), 设定"财报后第一分钟若 -5% 就停止减仓"。财报后跌 3%, 他按计划卖了一半。一周后复盘: 计划 vs 执行匹配, 他给自己一个 +1。**他得到的不是预测, 是纪律**。

### 场景 B: 港股突然跳水, 下午 3 点
下午 2:45 港股指数 -4%, 他手上 3 只港股持仓。agent 弹出一张 dashboard: "你 3 只港股持仓合计占组合 11%, 今日账面 -2.8%; 你上月定的'单国家暴露上限' = 15%, 仍在阈值内; 你订阅顾问最近两期对港股的立场: 中性; 历史相关性: 你这 3 只和港股整体相关性 0.82 —— 今日跳水主要不是 alpha, 是 beta; 你上次在类似行情(2025-11 港股 -5%)的决定是 '不动 + 第二天加仓', 事后回报 +7%。" 他看完, 决定**不做任何操作**, 并给自己写一句话: "2026-04-24: 港股盘中跳水, 未触发规则, 遵守上次经验, 选择不动。" **他得到的不是即时行动, 是不被情绪带走的克制**。

### 场景 C: 半年回顾, 一个周六的晚上
周六晚上, 孩子睡了, 他泡了杯茶, 打开 agent 的"半年回顾"视图。它给他看一份 15 页的 PDF: **你这 6 个月的 42 次决策里, 写了 pre-mortem 的 27 次平均 return +3.2%, 没写的 15 次平均 -1.1%; 你最准的判断是 TSM 的多头 thesis (4 次加仓全正); 你最错的判断是 BABA 的翻身押注 (3 次加仓, 全都止损); 你订阅的那位顾问全部 24 期观点里, 和你操作一致的 18 次平均跑赢基准 2%, 不一致的 6 次你自己的判断更对 4 次。** 他开始看见一个更诚实的自己: 他比自己想象的更擅长半导体, 比自己想象的更不擅长互联网翻身股; 顾问比自己强, 但他对某些板块有顾问没有的微弱 edge。他对妻子说: "下半年我计划把 BABA 类的押注砍掉, 把精力放在半导体和电力能源上。" **他得到的不是更多收益, 是更清楚的自我认知**。

## 4. Natural extensions (v0.1 之后 2 年内)

1. **"顾问分发基础设施"**: 从个人工具升级成顾问的分销工具。那位哥大顾问每周写一次观点, 通过 agent 分发给他 n 个付费订阅者, 每个订阅者得到"按你组合定制"的版本。这把产品从"工具"升级成**"专家 → 粉丝"的中间层**, 顾问和用户都受益, agent 厂商抽成或订阅。

2. **"决策档案市场 / 跟买跟卖的反面"**: 用户可选择公开自己的匿名化决策档案(仓位比例而非具体金额), 供同圈人匿名参考。不是跟单 —— 是看"相似背景的人怎么想"。长期沉淀出一个**中年技术人的决策共识层**, 比 Reddit 的噪声版本高一个量级。

3. **"家庭视角扩展"**: 从"个人操盘档案"升级成"家庭财务档案"—— 加入配偶收入/支出/保险/教育金维度。这正好对上 L1 菜单的 Direction 5(家庭首席资本官), 但是**从 v0.1 的 trader 视角有机演化出来的**, 而不是先从家庭视角切入。

4. **"多顾问 / 多信源聚合"**: v0.1 只支持一个订阅源, v1.0 支持 3-5 个, 让用户看到"多顾问共识度/分歧度", 弱化单点依赖。对应 L1 菜单 Direction 1 的一种实现。

5. **"事件驱动的主动提醒"**: v0.1 被动响应, v0.5 主动提醒 —— 你持仓股有财报/分红/重大新闻时, agent 提前 24 小时推一条"该决策的事件已近"。像日历通知, 但知道"这件事对你哪些仓位有影响"。

## 5. Natural limits (这个 idea 不应该是的样子)

- **不应该承诺 alpha**。这个产品不该说"我帮你跑赢市场"—— 它该说"我帮你更诚实地投资"。一旦跑进 alpha 承诺, 就会被拉进 robo-advisor 的红海, 比不过 Wealthfront/Magnifi 的规模和合规。
- **不应该做自动下单**。合规/信任门槛陡升, 且违背 core value("让你成为更好的决策者"而非"替你做决策")。永远保留"人类最后一按"。
- **不应该扩展到完全不懂金融的小白用户**。那群用户要的是 robo-advisor("我不想学, 你帮我搞定"), 不是 pre-mortem("我要成为更好的投资者")。产品语气、UX、学习曲线都需要为"有判断力+没时间+想变强"的人服务。
- **不应该扩展到期权/加密/高杠杆/日内交易**。这些场景需要分钟级响应、完全不同的决策节奏, 和"晚上 20 分钟 + 周末 1 小时"的使用节奏冲突。扩进去等于失焦。
- **不应该做成社交产品**。L1 Direction 7 的"同代参照"诱惑很强, 但一旦社交化, 会被 Reddit/Twitter 的舆论场拉走 core user —— 他们来这是为了**从噪声里安静下来**, 不是加入另一个噪声池。
- **地理限制**: 这个产品对"覆盖三市场(美股/港股/A股) + 订阅单个华语投顾"的中年华人技术移民/高管是完美 fit。扩到纯美国用户或纯中国大陆用户都会失去核心 anchor。这是一个**特定文化/语境的产品**, 不是 global。
- **时间限制**: 这个产品的市场窗口, 某种意义上和"中年技术人对未来不确定感"这个时代情绪绑定。如果 10 年后经济周期回暖、这种焦虑消退, 产品的 ruling emotion 会减弱。但短中期(未来 5-10 年), 窗口稳定。

## 6. Three honest questions about this idea (seed L2R2 search)

1. **中年技术人真的有这种"把人类顾问作为 anchor 但用 AI 放大"的需求吗?** 还是我(一个 AI)在自动夸大"技术人会想用工具"这件事? 搜: 付费订阅投顾的用户实际流失率、用户在论坛上表达"我想要 X 但没有"的帖子、已有工具(如长桥/雪球的订阅功能)的实际 engagement。

2. **"决策 pre-mortem + post-mortem 的日记"是不是本质上一个已经被反复证明会被用户放弃的产品形态?** L1 Direction 2 的验证已经揭示"几周后就停了"的失败模式很强。到 L2 这层应该更细地挖: 是哪种 journal 会被放弃? 是否有任何 trading journal 实现了 >50% 半年后保留率? 如果有, 他们做对了什么?

3. **"为付费人类投顾做一个 AI 放大器"这个商业结构, 是不是一个双赢但受法律法规严重约束的产品形态?** 比如订阅者数据隐私、顾问的 fiduciary liability、个性化建议 vs 通用建议的监管边界。虽然具体合规是 L3/L4 的事, 但现在就应该把"监管是否让这个结构根本不可行"这个天花板探一探 —— 如果在这层有硬限制, 整个 idea 需要重新 reframe。
