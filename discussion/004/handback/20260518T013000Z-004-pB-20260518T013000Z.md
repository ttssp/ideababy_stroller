## В§7 Gotchas / lessons learned

1. **OpenAI SDK ејӮеёёдҪ“зі»**:`APIConnectionError` / `APITimeoutError` / `RateLimitError` / `InternalServerError` / `ConflictError` йғҪз»§жүҝ `openai.OpenAIError` В· дёҚжҳҜ builtin `ConnectionError` / `TimeoutError`гҖӮretry catch еҝ…йЎ»жҳҫејҸ importгҖӮ
2. **OpenAI Whisper duration**:й»ҳи®Ө `response_format` еҸӘиҝ” text В· еҝ…йЎ»дј  `response_format="verbose_json"` жүҚиҝ” `duration`(D-spec-5a cost cap зңҹдҫқиө–)гҖӮ
3. **APIStatusError predicate**:408 Request Timeout зңҹ transient В· дҪҶ BadRequestError/AuthenticationError/PermissionDeniedError/NotFoundError/UnprocessableEntityError йғҪжҳҜ APIStatusError еӯҗзұ» В· еҝ…йЎ» predicate `status_code==408` йҡ”зҰ»гҖӮ
4. **retry иҫ№з•ҢдёҖиҮҙ pattern**:LLMClient.\_record_usage зңҹи·Ҝеҫ„ try/except + log дёҚйҳ»ж–ӯ(L407-411)В· T022 и·ҹе®ғеҜ№йҪҗ(round 5 F1 fix)В· йҳІ partial-failure йҮҚеӨҚд»ҳиҙ№дёҠжёё APIгҖӮ
5. **partial-failure recovery зңҹи·Ҝеҫ„**:еҸҢйҳ¶ж®ө(transcribe + extract)дёӯй—ҙ transcript жІЎжҢҒд№…еҢ– вҶ’ д»»дҪ•дёҖйҳ¶ж®өеӨұиҙҘ вҶ’ caller retry йҮҚж–°и°ғ WhisperгҖӮзңҹи·Ҝеҫ„йңҖ audio hash вҶ’ transcript cache(transcript_cache.py В· и·Ё T022/T030 иҫ№з•Ң В· з•ҷ FU-T022-followup-1)гҖӮ
6. **LLMCache key shape**:cache key дёҚеҗ« prompt body В· д»… (advisor_week_id, ticker, template_version, model, extras) В· KeypointExtractor еҝ…йЎ»жҠҠ `transcript_sha256_16` еЎһ cache_key_extras йҳІеҗҢ week дёҚеҗҢ audio дёІ(round 1 F3)гҖӮ
