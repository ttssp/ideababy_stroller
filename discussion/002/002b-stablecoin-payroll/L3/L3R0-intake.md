## Block 7 — Freeform(operator 自述约束,**最重要的一批信号**)

operator 主动提了四条 —— 每一条都直接影响 v0.1 shape:

### 💡 "我住亚太给欧美客户"
- 含义:**时差是硬约束**(亚太上午 = 欧美前夜)
- 影响 scope:concierge 的"24 小时异步响应"不是卖点是必须(你醒客户睡,客户醒你睡)
- 影响 cohort:美国本土 founder 客户期望 9-5 响应,你给不了;更适合**欧洲(时差小)+ 跨时区已经习惯异步的 DAO / crypto-native founder**

### 💡 "我无 CPA 资质"
- 含义:**心智定位天花板封顶在 bookkeeper / people ops 档,CFO-adjacent 档不能去**
- 这直接仲裁 L2 的 Opus/GPT 分歧:**GPT 胜**(心智 = bookkeeping-only,不做 CFO 味)
- 天花板从 Opus 的 $10-15k/月 operator 下调到 GPT 说的"先证 wedge"
- 定价从 $480/月起步,能到 $600-800/月 已是上限(除非未来补 CPA)

### 💡 "没有已存 DAO 网络"
- 含义:**cohort funnel 是 v0.1 最大障碍**
- 小 DAO ops 是证据最强的 cohort,但 operator 没通路——硬冲突
- L3 必须产出一条 candidate **"0 网络如何 3 月内获 10 客户"** 的具体路径(否则 cohort 选择无效)
- 可能的选项:Discord cold-reach / Farcaster 公开 thread / Superteam / YC 校友网络 / Request Finance 用户共情群 / 通过 002g(DAO bounty)接单时自然接触 ops 角色

### 💡 "没什么虚拟币经验"
- 含义:**operator ground truth 在 crypto 操作栈几乎 0**(Safe / multi-sig / multi-chain / gas / approval / bridging)
- 这是**最大的隐藏风险**——concierge 的底层价值是"帮客户把 SOP 跑顺",但 operator 自己要先学会 SOP
- L3 必须正面回答:**前 X 周投入多少小时学 crypto 操作栈** + **是否先做 beta 客户自己当第一个客户**
- 影响时间预算:2-3 月 MVP 前可能需要**追加 2-4 周 ramp-up**(或把 ramp-up 折进 2-3 月内但 scope 对应缩减)
